# Increment.gg

## Development Setup

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/increment.gg.git
   cd increment.gg
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Build the assets:
   ```
   npm run build
   ```

4. Start the development server:
   ```
   npm run dev:all
   ```

5. Open your browser and navigate to `http://localhost:3000`
